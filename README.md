# Tic Tac Toe Game

A modern and interactive Tic Tac Toe game designed with a sleek purple-themed interface. Players take turns marking spaces on a 3x3 grid to achieve victory or a draw.

## Features

- **Two-player functionality**: Play against a friend in a turn-based game.
- **Winning and draw detection**: Alerts the players if there's a win or a draw.
- **Reset functionality**: Start a new game with a single click.
- **Accessible design**: Includes keyboard focus styles and ARIA labels for accessibility.
- **Responsive UI**: Adjusts seamlessly to different screen sizes for a great experience on desktop and mobile devices.
- **Modern animations**: Subtle hover and pop effects enhance the user interaction.

## Technologies Used

- **HTML**: Defines the structure of the game board and layout.
- **CSS**: Implements a visually appealing purple theme using CSS variables, along with responsive and accessible design elements.
- **JavaScript**: Handles the game logic, including turn-taking, win conditions, and reset functionality.

## How to Play

1. **Start the game**: The first player is assigned "X", and the second player gets "O". The game will indicate whose turn it is.
2. **Mark a space**: Click any empty cell to place your marker (X or O).
3. **Winning the game**: Be the first to get three of your markers in a row (horizontal, vertical, or diagonal).
4. **Game reset**: Click the **Reset Game** button to clear the board and start fresh.
